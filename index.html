🛡️ Official Poki Purity Laws

Miki eliminates old legacy bloat to safeguard physical hardware real-time systems.

🦠 0% Malware

Every C-driver undergoes strict manual moderation and a Poki static virus scan before publication. No dangerous triggers allowed.

⏳ The 1-Year Law

If a library is abandoned and stops supporting newer versions of Miki, it stays for 1 year. If no updates are pushed, it is permanently deleted.

💸 100% Free

All submissions must be completely free and open-source. Paid or closed-source packages are strictly forbidden in this registry.

💻 Supported Languages

You can write Miki ecosystem libraries strictly in:

  • Miki (Native source code for algorithms and mathematics)
  • Pure C (Low-level hardware drivers / wrappers)

🔒 Authorship & Copyright

Your published library belongs 100% to you. Nobody can claim your authorship or alter your code maliciously. Only the verified original author can release updates, patches, and patches using cryptographic security keys.
